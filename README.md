# ciudades_conocimiento_tesis_dpp_cide
Ciudades del Conocimiento: Metrópolis en Transición en un País Emergente (México, 2003-2018) Este repositorio contiene los materiales de investigación generados como parte de mi tesis doctoral. Incluye las bases de datos originales, los códigos de procesamiento en R, las gráficas y los tabulados finales utilizados en el estudio.
